Projects
Cloudvision

Cloudvision is a mobile app that allows users to recognise cloud species by taking a photo. I used Flutter for this app, and made use of TensorFlow machine learning for the cloud recognition feature. It was made as part of an entry for the Science Talent Search Victoria 2021.

Pogbot

Pogbot is a multi-purpose Discord bot created while experimenting with the Discord.py library. Features include competitions, leaderboards, quote generator, audio player and a Python compiler.

GitHub

Edge of Time

Edge of Time was a game created for the VimJam 2 game jam - a game making competition where you have limited time to make a game based on a theme. I worked with a friend to complete this (hence the amazing pixel art) and it turned out great considering the timeframe. We used the Unity game engine to create this.
