About Crest

Crest is a web framework written in Python and designed for simplicity and efficiency. We always look to introduce features that enhance the experience for the developer and the user. While based on the popular model-template-view structure, the uses of Crest are not limited to websites and web apps. Crest can be used to build fullstack apps or backend APIs with the API page modules included.
