Theory

We have covered some theory relating to images. In this page, I will discuss the following concepts:

  • Image resolution
  • Image formats
Note: all images are in the public domain

Image resolution

Resolution is a word used to describe the quality of an image. It is often measured with a ratio, such as DPI.

DPI or "dots per inch" is a way to measure the resolution of the image. The DPI will be equal to the number of "dots" (pixels) per inch in the image. Below is the equation for determining the DPI of an image.

$$ {DPI =\frac{pixels\;wide}{inches\;wide}} $$

The standard resolution for photographs and images is 300dpi. 1200dpi is an excellent resolution for high-detail images. You may also come across images that are 600dpi or 72dpi.

Image formats

When we talk about image formats, we can usually classify them as either vector or raster images. Vector images image data in the form of an equation. This is great for images with low detail such as company logos or icons, as you will not lose any quality when resizing.

Raster images store image data in a much more raw form. They store the RGB values of every pixel in the image. In a bitmap image, every single pixel is stored, but other formats compress the image, so that only certain parts of the image are stored, and patterns can be used to evaluate the colour and position of other pixels. Raster images are ideal for most images such as photographs or graphics. The downside of this image type is that it can appear pixelated when resized.



Common raster image formats:


1. RAW
RAW is a lossless (uncompressed) image format commonly used in the photography industry. Many cameras capture RAW images by default. It does have a high file size, but it maintains a lot of quality.


2. TIFF
TIFF stands for Tag Image File Format. It is a lossless image format commonly used for photography. It is an industry favourite for storing high-quality images.


3. PNG
PNG is one of the most common image formats in the world. It stands for Portable Network Graphics. It is a lossless image format used most often for digital graphics. It is supported by almost all software and features very small file sizes, making it a very popular format.


4. JPG
JPG is a lossy image format used commonly for digital images. It is the most common format in digital cameras, operating systems and on the internet. Similarly to PNG, it does not have a large file size.


5. GIF
GIF is a lossy image format often used for animated images. It can store multiple frames of short animations and videos, making it useful for demonstrations on websites. Unfortunately, it has a large file size and sacrifices a lot of quality.


6. PDF
PDF is an image format usually used for documents and printing. It typically has a medium-large file size. PDF is not always referred to as an image format, as it stores data of text, images, etc. as raw binary data, making it more of a document format.


7. PSD
PSD (Photoshop Document) is the default document format for Photoshop projects. It stores layers and details about individual graphics within an image so that it can be edited inside Photoshop when opened. A PSD file is not typically used for images that are not worked on in Photoshop.
